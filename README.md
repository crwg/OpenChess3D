## ✨ Features

- 🏆 **Play vs AI** – Challenge Lozza, a strong open‑source chess engine.
- 📜 **PGN support** – Import, export, and analyse your games in Portable Game Notation.
- 🎮 **3D graphics** – Fully rotatable board with realistic pieces and lighting.
- 🆓 **100% free** – No ads, no paywalls, no subscriptions. Forever.
- 🧩 **Open source** – Built with love and available for everyone.
